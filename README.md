# Caderno Temático - Fundamentos de Agentes de IA

## Sobre o Projeto

Este projeto foi desenvolvido como parte do desafio da DIO utilizando o NotebookLM como ferramenta de estudo assistida por Inteligência Artificial.

O tema escolhido foi "Fundamentos de Agentes de IA", devido à crescente utilização de sistemas inteligentes capazes de tomar decisões, automatizar tarefas e interagir com usuários de forma natural.

## Objetivos de Estudo

* Compreender o conceito de agentes inteligentes.
* Entender a arquitetura de um AI Agent.
* Aprender sobre LLMs (Large Language Models).
* Conhecer ferramentas para construção de agentes.
* Explorar aplicações práticas em automação e assistentes virtuais.

# Dificuldades Encontradas

* Algumas respostas eram muito genéricas.
* Foi necessário reformular perguntas para obter respostas mais técnicas.
* Prompts muito amplos geravam informações superficiais.

### Exemplo de melhoria

Prompt inicial:

Como funciona IA?

Prompt melhorado:

Explique detalhadamente como um agente de IA utiliza um LLM para interpretar comandos do usuário e executar ações externas.

# Miniguia de Estudo

## O que é um Agente de IA?

Um agente de IA é um sistema capaz de perceber informações, processá-las, tomar decisões e executar ações para atingir objetivos específicos.

## Componentes Principais

* Entrada de dados
* Processamento
* Modelo de IA
* Memória
* Ferramentas externas
* Saída

## Aplicações

* Chatbots
* Assistentes virtuais
* Automação empresarial
* Sistemas de recomendação
* Suporte ao cliente

---

# Glossário

## AI Agent

Sistema inteligente capaz de agir de forma autônoma.

## LLM

Large Language Model. Modelo treinado para compreender e gerar linguagem natural.

## Prompt

Instrução fornecida para a IA.

## API

Interface que permite comunicação entre sistemas.

## Automação

Execução automática de tarefas sem intervenção humana constante.
